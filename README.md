# همیار

فرانت‌اند فارسی و راست‌به‌چپ با یک کانتینر موبایل‌محور که در دسکتاپ هم در وسط صفحه می‌ماند.

## راه‌اندازی

1. فایل `.env` بسازید و این مقدارها را داخلش قرار دهید:

```env
VITE_SUPABASE_URL=https://your-project.supabase.co
VITE_SUPABASE_ANON_KEY=your-anon-key
```

2. وابستگی‌ها را نصب کنید:

```bash
npm install
```

3. برنامه را اجرا کنید:

```bash
npm run dev
```

## صفحات

- `/` صفحه اصلی با یک قالب ساده و عنوان پروژه
- `/admin` صفحه ورود ادمین با فرم لاگین و نمایش پروفایل بعد از احراز هویت
